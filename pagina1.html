CONCEPTOS BASICOS

  • Sitio web: Conjunto de páginas web organizadas bajo un mismo dominio y con una estructura común.
  • Página web: Un solo documento dentro de un sitio web, accesible a través de un navegador.
  • Hipervínculo: Enlace que permite navegar entre páginas web o recursos en internet.
  • URL (Uniform Resource Locator): Dirección única que identifica una página o recurso en la web.
  • HTML (HyperText Markup Language): Lenguaje de marcado que estructura el contenido de una página web.
  • CSS (Cascading Style Sheets): Lenguaje que define la apariencia y el diseño de una página web.
  • JS (JavaScript): Lenguaje de programación que permite interactividad en sitios web.
  • DNS (Domain Name System): Sistema que traduce nombres de dominio en direcciones IP.
  • IP (Internet Protocol): Dirección numérica que identifica un dispositivo en una red.
  • Dominio web: Nombre único que identifica un sitio en internet (ej. google.com).
  • Cliente-servidor: Modelo de comunicación donde un cliente solicita datos y un servidor los entrega.
  • FTP (File Transfer Protocol): Protocolo para transferir archivos entre computadoras en una red.
  • HTTP (HyperText Transfer Protocol): Protocolo para la transferencia de información en la web.
  • HTTPS (HTTP Secure): Versión segura de HTTP que cifra los datos con SSL/TLS.
  • Frontend: Parte visual e interactiva de un sitio web con la que el usuario interactúa.
  • Backend: Lógica y procesamiento de un sitio web que ocurre en el servidor.
  • Hosting: Servicio que almacena y hace accesible un sitio web en internet.
  • Hosting Privado Virtual (VPS): Servidor virtual dedicado dentro de un servidor físico, con recursos exclusivos.
  • Hosting en la nube (Cloud Hosting): Uso de múltiples servidores interconectados para mayor escalabilidad y rendimiento.
  • W3C (World Wide Web Consortium): Organización que establece estándares para la web.
